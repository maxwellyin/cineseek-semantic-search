Retrieval System

Search movies by meaning — not keywords.

CineSeek is a semantic movie search system that understands natural-language queries. It combines embedding-based retrieval, FAISS-powered candidate search, and an optional LLM agent for reranking and explanation.

System Snapshot

Retrieval-first architecture, with an optional LLM agent.

The system keeps retrieval explicit and efficient: sentence embeddings, FAISS-based nearest neighbor search, and a lightweight dual-tower model. An optional LLM agent improves ranking and adds explanations, without replacing the underlying retrieval pipeline.

What It Does

Discover movies from natural-language descriptions.

Semantic Search

Queries and movies are mapped into a shared embedding space, enabling retrieval beyond exact keyword matches.

Fast Retrieval

FAISS enables low-latency nearest neighbor search, making the system responsive for interactive use.

Agent Refinement

An optional LLM agent refines queries, reranks results, and provides concise explanations.

Live System

What happens after you submit a query

In the live system, queries are embedded and matched against a prebuilt FAISS index. An LLM agent then refines the results through reranking and explanation.

1. Retrieve

The query is embedded and matched against a FAISS index of movie representations.

2. Rerank

An LLM agent reorders candidates and resolves vague or ambiguous queries.

3. Explain

The system provides short explanations to make results interpretable and transparent.
